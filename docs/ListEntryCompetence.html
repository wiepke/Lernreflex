Class: ListEntryCompetence

ListEntryCompetence

Holds the markup for several list elements ocurring on different views in this app. To render a row, you need to pass a property rowData containing type = {one of the possible function names} without a leading _ In the future the markup for a row could be transfered back into the belonging class, to make things more consistent
